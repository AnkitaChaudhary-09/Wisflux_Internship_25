<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

------explanation-------
<!DOCTYPE html> – Declares the document type.

<html> – Root element of the webpage.

<head> – Contains meta info like title, CSS, etc.

<title> – Title shown in browser tab.

<body> – Visible content on the web page.

Common HTML Tags
Tags          Purpose                                      Examples
<h1> to <h6>	Headings (<h1> is biggest, <h6> smallest)	 <h1>Main Title</h1>
<p>	            Paragraph                                    <p>This is a paragraph.</p>
<a>	            Anchor (links to other pages)	             <a href="https://google.com">Google</a>
<img>	        Image	                                     <img src="image.jpg" alt="My Image">
<ul> & <ol>	    Unordered & Ordered list	                 <ul><li>Item</li></ul>
<li>	        List item	                                 <li>Milk</li>
<br>	        Line break (new line)	                     Line 1<br>Line 2
<hr>	        Horizontal rule (line)	                     <hr>
<div>	        Container used to group elements	         <div>Some content</div>
<span>	        Inline container for text styling	         <span style="color:red">Red Text</span>
<strong>	    Bold text	                                 <strong>Important</strong>
<em>	        Italic text	                                 <em>Important</em>
<a>	            Creates a link to another page or website	 <a href="https://google.com">Google</a>
<img>	        Displays an image	                         <img src="img.jpg" alt="My Image">
<audio>	        Embeds audio content	                     <audio controls><source src="song.mp3"></audio>
<video>	        Embeds video content	               <video width="320" height="240" controls><source src="movie.mp4"></video>
